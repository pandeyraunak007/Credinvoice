# CredInvoice

A digital invoice discounting platform connecting MSME Sellers with Financiers (Banks/NBFCs), enabling sellers to raise working capital quickly while giving financiers verified investment opportunities.

## Phase 1 - Reverse Factoring

The financier funds MSMEs against invoices provided by Buyers. Buyers get discounts for releasing payments early.

### User Roles

- **Seller (MSME)**: Onboard via KYC, upload invoices, receive financing offers
- **Financier (Bank/NBFC)**: Review sellers, approve credit limits, bid on invoices
- **Admin**: Full platform control, manage KYC, disputes, compliance

### Key Features

- Invoice Upload & Management
- KYC Workflow
- Credit Limit Management
- Bidding System
- Disbursement Tracking
- Dispute Resolution
- Role-Based Access Control
- 2-Factor Authentication
- Full Audit Logging

## Future Phases

- **Phase 2**: Invoice Funding
- **Phase 3**: NBFC, Lend by CredInvoice Books
